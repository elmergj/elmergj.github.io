# Elmer Gonzalez — Personal Portfolio

Welcome to my personal portfolio project. This repository contains my customized version of a portfolio template, adapted with my own content, design adjustments, and personal branding.

## 👤 About Me

My name is **Elmer Gonzalez**, and this project represents my personal portfolio where I showcase my work, skills, and experience as a developer.

## 🧩 Template Origin

This project is based on the following template:

https://github.com/rishikesh2003/my-portfolio

The original structure and foundational components come from this repository, but the content, styling, and modifications have been adapted for my personal use.

## 🚀 Tech Stack

This project is built using:

* **Astro**
* **React**
* **Tailwind CSS**

## 📦 Getting Started (Local Development)

To run this project locally:

```bash
pnpm install
pnpm run dev
```

Then open your browser at the local development URL (usually `http://localhost:4321`).
